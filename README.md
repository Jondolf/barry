# Barry

**Barry** is a fork of **Parry**, providing a set of 2 and 3-dimensional geometric and collision detection libraries.
These libraries are `barry2d`, `barry3d`, `barry2d-f64`, and `barry3d-f64`. It is forever free
and open-source!
